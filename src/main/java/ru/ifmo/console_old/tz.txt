задачи модуля:
генерировать запросы, принимать и верифицировать ответы в соответствии с xml файлом,
 в котором описаны команды, их аргументы и ограничения.
 нужна возможность выбирать 1 из эквивалентных параметров используя xml файл.
 на выходе выдает json, в котором указаны названия аргументов и их значения.
 нужна воможность гибко переопределять верификацию для каждого параметра и команды в целом, однако необходима стандартная реализация базовых случаев.
 в xml указывается, если верификатор переопределен.

 каждая команда имплементирует интерфейс


у нас есть драйвер определенного устройства ввода вывода, с помощью него то что раньше было ConsoleIO полуает возможность взаимодействовать с дпругими устройствами ввода.
